C:
1. Неправильный размер памяти в realloc
   В add_grade: realloc(student->grades_count) вместо realloc(student->grades_count + 1)
   Не хватало места для новой оценки

2. Ошибка в формуле расчета среднего балла
   Было: (average + grade) / count
   Стало: сумма всех оценок / количество
   Формула показывала 2.78 вместо 4.17

3. Выход за границы массива в циклах
   В add_grade и find_student: i <= count вместо i < count
   Читал память за пределами массива

4. Возможное переполнение буфера имени
   strcpy вместо strncpy
   Длинное имя могло переполнить буфер

5. Поиск несуществующих студентов
   В find_student: i <= capacity вместо i < count
   Искал студентов которые не были добавлены

6. Отсутствие защиты после освобождения памяти
   После free(manager->students) не обнулял указатель
   Возможное использование после освобождения

java:
1. Неправильная проверка при снятии средств
   Было: if (balance < 0)
   Стало: if (balance < amount)
   Разрешал снять больше денег чем есть на счете

2. Ошибка в начислении процентов
   Было: deposit(balance + interest)
   Стало: balance += interest
   Удваивал деньги вместо начисления процентов

3. Небезопасный порядок операций при переводе
   Было: положить → снять
   Стало: снять → положить
   Деньги терялись если снятие не удавалось
